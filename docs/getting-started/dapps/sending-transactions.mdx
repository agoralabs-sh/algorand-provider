# Sending Transactions

* [Overview](#overview)

## Overview

Wallets, if supported, can commit transactions to the network using their connection to a node.

## Sending a transaction with the default wallet

If you only want to send transactions to the network with the default wallet, you can simply call:

<Tabs
  defaultValue="javascript"
  values={[
    { label: 'Javascript', value: 'javascript' },
    { label: 'TypeScript', value: 'typescript' },
  ]}>
  <TabItem value="javascript">

```js
try {
  const result = await window.algorand.postTxns({
    stxns: [
      'gqNzaWfEQ...',
    ],
  });

  console.log(result);
  /*
  {
    id: 'awesome-wallet',
    txnIDs: [
      'OKU6A2Q...',
    ],
  }
  */
} catch (error) {
  // handle error
}
```

  </TabItem>
  <TabItem value="typescript">

```typescript
import type { IBaseResult, IPostTxnsResult } from '@agoralabs-sh/algorand-provider';

try {
  const result: IBaseResult & IPostTxnsResult = await window.algorand.postBytes({
    stxns: [
      'gqNzaWfEQ...',
    ],
  });

  console.log(result);
  /*
  {
    id: 'awesome-wallet',
    txnIDs: [
      'OKU6A2Q...',
    ],
  }
  */
} catch (error) {
  // handle error
}
```

  </TabItem>
</Tabs>

:::caution

If this operation is not supported, then a [`WalletOperationNotSupportedError`](../../api-reference/errors#walletoperationnotsupportederror) will be thrown.

:::

## Sending a transaction with a specific wallet

If you want to target a specific wallet to send the transactions, you can simply pass the ID of the wallet to the options object:

<Tabs
  defaultValue="javascript"
  values={[
    { label: 'Javascript', value: 'javascript' },
    { label: 'TypeScript', value: 'typescript' },
  ]}>
  <TabItem value="javascript">

```js
try {
  const result = await window.algorand.postTxns({
    id: 'another-awesome-wallet',
    stxns: [
      'gqNzaWfEQ...',
    ],
  });

  console.log(result);
  /*
  {
    id: 'another-awesome-wallet',
    txnIDs: [
      'OKU6A2Q...',
    ],
  }
  */
} catch (error) {
  // handle error
}
```

  </TabItem>
  <TabItem value="typescript">

```typescript
import type { IBaseResult, IPostTxnsResult } from '@agoralabs-sh/algorand-provider';

try {
  const result: IBaseResult & IPostTxnsResult = await window.algorand.postTxns({
    id: 'another-awesome-wallet',

      'gqNzaWfEQ...',
    ],
  });

  console.log(result);
  /*
  {
    id: 'another-awesome-wallet',
    txnIDs: [
```

  </TabItem>

:::caution

:::
